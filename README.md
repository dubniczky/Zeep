## Usage

### Build

Build single version for current system

```bash
make build
```

Build multiple versions for different pre-specified systems 

```bash
make publish
```

### Run

Run tests

```bash
make run
```

Run executable directly

```bash
cd bin
./zeep [FILE_LIST]
```
